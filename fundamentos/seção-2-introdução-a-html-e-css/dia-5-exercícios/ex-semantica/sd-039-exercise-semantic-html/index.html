Fatos sobre o Stomatopoda

  1. Possuem uma visão incrível;

  2. Possuem um soco poderoso;

  3. São exclusivamente carnívoros;

  4. O segundo par de patas é muito desenvolvido.

Informações Científicas

Seu nome científico é Odontodactylus scyllarus.

Sua classificação científica é Animalia Arthropoda Crustacea Malacostraca Hoplocarida Stomatopoda Latreille, 1817

Eles enxergam muitas cores

Os estomatópodes possuem um total de 16 cones fotoreceptores, isso são 13 a mais do que nós seres humanos.
